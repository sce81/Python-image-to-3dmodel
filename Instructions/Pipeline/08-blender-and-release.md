# Blender and release

Run Blender only after mesh acceptance. It may normalize axes, dimensions, valid microscopic debris, and export compatibility; it must not invent missing structure.

Require final PBR review, contract compliance, and explicit user visual approval before writing UE-ready GLB and Blend files under `Outputs`.

After changes to either generator or Blender script, run the proportional smoke test when practical and report the final filenames.
