# TRELLIS 2 texturing

Texture only an accepted extracted geometry mesh.

Use Microsoft `Trellis2TexturingPipeline` with the paired texture render and the unchanged accepted geometry. Keep the texture raster camera-aligned with the geometry raster. Do not claim a one-pass `pipe.run()` output using the clay raster is final texture quality.

Require a 4096 atlas for final review and inspect paint, glazing, lamps, trim, wheels, grille, and map inventory.
